
=================================
xdc3.js - Ethereum JavaScript API
=================================

xdc3.js is a collection of libraries that allow you to interact with a local or remote ethereum node using HTTP, IPC or WebSocket.

The following documentation will guide you through :ref:`installing and running xdc3.js <adding-xdc3>` as well as providing an :ref:`API reference documentation <#id1>` with examples.
